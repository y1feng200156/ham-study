# 偶极子天线 (Dipole Antenna)

## 1. 概述
偶极子天线（Dipole）是天线界的“元祖”，也是所有线天线的基础。
* **结构**：由两根等长的导体组成，在中心断开并馈电。
* **最常用形态**：**半波偶极子 (Half-wave Dipole)**，即总长度 $L = \lambda/2$。

## 2. 工作原理：驻波 (Standing Wave)
当射频电流馈入天线时，电流在导线末端反射，与入射波叠加形成**驻波**。
* **对于半波偶极子 ($L=\lambda/2$)**：
    * **两端**：电流为 0 (波节)，电压最大 (波腹)。
    * **中心**：电流最大 (波腹)，电压最小 (波节)。
    * **阻抗**：由于 $Z = V/I$，中心点电压低、电流大，因此阻抗较低，约为 **73Ω** (纯电阻性)。

## 3. 辐射方向图公式 (Mathematical Model)
对于中心馈电的偶极子，其远场辐射方向图 $F(\theta)$ 仅取决于导线长度 $L$。

**通用公式**（适用于任意长度 $L$）：
$$F(\theta) = \frac{\cos(\frac{kL}{2} \cos \theta) - \cos(\frac{kL}{2})}{\sin \theta}$$
其中 $k = 2\pi/\lambda$ 是波数，$\theta$ 是与导线轴线的夹角。

**半波偶极子特例** ($L=\lambda/2 \Rightarrow kL/2 = \pi/2$)：
代入上式，由于 $\cos(\pi/2)=0$，公式简化为经典的：
$$F(\theta) = \frac{\cos(\frac{\pi}{2} \cos \theta)}{\sin \theta}$$

* **形状**：在二维剖面（E平面）上呈**“8”字形**。
* **最大辐射**：当 $\theta = 90^\circ$ 时（垂直于导线方向），$F(90)=1$，辐射最强。
* **零点**：当 $\theta = 0^\circ$ 或 $180^\circ$ 时（导线轴线方向），$F=0$，无辐射。

## 4. 为什么不用全波长偶极子？
如果你在模拟器中将长度设为 $1.0\lambda$，你会发现：
1.  **方向图**：虽然变窄了一点，但依然是 8 字形，增益略有提升。
2.  **阻抗灾难**：在中心点，全波长天线的电流为 0（波节），电压极高。这意味着输入阻抗 **Z 趋近于无穷大**（数千欧姆）。
3.  **结论**：50Ω 的电台无法直接驱动全波长偶极子。如果非要用全波长，通常使用**端馈**（即 EFHW 的原理）或特殊的匹配网络。

## 5. 阻抗特性
* **73Ω**：理想自由空间中的半波偶极子。
* **50Ω**：当偶极子被做成“倒V”形状（夹角 90-120度）时，阻抗会降低到 50Ω 左右，这正是倒V天线流行的原因——省去了巴伦的阻抗变换功能，只需要 1:1 巴伦做平衡-不平衡转换即可。
