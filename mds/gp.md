# GP天线 (Ground Plane Antenna)

### 1. 什么是 GP 天线？
GP 天线（Ground Plane），即**地网天线**，属于垂直单极天线（Monopole）的一种。
* **结构**：一根垂直的振子（Radiator）竖在中间，底部有 3-4 根（或更多）金属杆水平或向下倾斜伸出，称为**地网**（Radials）。
* **长度**：垂直振子和地网通常都是 **1/4 波长** ($\lambda/4$)。

### 2. 核心原理：镜像原理 (Image Theory)
* **少了一半？**：偶极子天线由两根 1/4 波长振子组成。GP 天线只有上面一根，下面那根去哪了？
* **地网的作用**：地网（或大地）充当了**“电磁镜子”**。它在地下产生了一个虚拟的镜像振子。
* **虚实结合**：实体的上半部分 + 虚拟的下半部分 = 完整的偶极子。

### 3. 辐射方向图公式 (Mathematical Model)
对于理想的 1/4 波长垂直单极子天线（架设在无限大完美导电地平面上），其归一化辐射场强公式为：

$$F(\theta) = \frac{\cos(\frac{\pi}{2} \cos \theta)}{\sin \theta}$$

其中 $\theta$ 是相对于天顶（垂直轴）的夹角。
* **当 $\theta = 0^\circ$ (天顶)**：$F(0) = 0$。说明正上方无辐射（盲区）。
* **当 $\theta = 90^\circ$ (地平线)**：$F(90) = 1$。说明沿地面方向辐射最强。
* **形状描述**：该公式描述的剖面图类似于一个横放的“8”字，或者两个相切的圆。

### 4. 地网角度的影响：下垂 vs. 上翘

GP 天线可以根据地网的角度调节其电气特性，主要分为两种形态：

#### 4.1 下垂地网 / 伞状 (> 90°)
* **形态**：地网向下倾斜，夹角通常为 **135°**。
* **阻抗**：从水平时的 36Ω **拉升至 50Ω**，完美匹配同轴电缆。
* **方向图**：保持低仰角特性（贴地发射），主波瓣指向地平线。
* **应用**：标准的短波/超短波 DX 天线。

#### 4.2 上翘地网 / W形 / 碗状 (< 90°)
* **形态**：地网向上翘起，像一个碗或开放的花朵。
* **阻抗**：从水平时的 36Ω **进一步降低**（可能降至 20Ω 以下）。通常需要配合加长振子或底部加感线圈来强制匹配。
* **方向图**：地网起到反射器的作用，将波束**向上托起**，导致仰角抬升。
* **应用**：
    * **城市通信**：利用高仰角跨越楼宇障碍。
    * **卫星通信**：主波瓣指向天空。

### 5. 辐射特性与优缺点 (通用)

#### 优点：
1. **低仰角发射 (Low Take-off Angle)**：这是垂直天线的杀手锏。信号主要贴着地面发射（仰角极低），非常适合 **DX（远距离跨国/跨洲）** 通信，因为信号能跳跃得更远。
2. **全向辐射 (Omni-directional)**：在水平面上360度无死角，不需要旋转器。

#### 缺点：
1. **噪音大 (Noisy)**：大多数人造噪音（电源干扰、点火噪声）都是垂直极化的，垂直天线很容易接收到这些噪音，听起来比水平天线（如八木、倒V）要吵。
2. **效率依赖地网**：如果是安装在地面的垂直天线，地网铺设得好坏直接决定了发射效率（地网损耗电阻）。架空的 GP 天线（Elevated Radials）效率相对较高。

**一句话总结**：GP 天线是“DX 利器”，用最简单的结构实现了低仰角发射和 50Ω 匹配，是基地台 V/U 段和短波 DX 通信的经典选择。
