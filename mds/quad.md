# 方框天线 (Loop Antenna)

### 1. 概述
方框天线（或环形天线）是由一圈导线组成的闭合回路。其形状可以是正方形、圆形、三角形或多边形。**尺寸决定了一切**，根据环的周长 ($C$) 与波长 ($\lambda$) 的比例，特性截然不同。

### 2. 电小环 (Small Loop) —— “磁偶极子”
* **定义**：$C < \lambda/10$
* **电流分布**：均匀恒定。
* **物理模型**：等效于磁偶极子。
* **方向图**：
    * 最大辐射方向：**沿着框面 (In Plane)**。
    * 形象比喻：“甜甜圈”套在框上。
* **缺点**：辐射电阻极低（几毫欧姆），效率极低，主要用于接收（如收音机磁棒）。

### 3. 全波谐振环 (Full-Wave Loop) —— “变形的双偶极子”
* **定义**：$C \approx 1\lambda$
* **电流分布**：形成正弦驻波。
* **物理模型**：等效于垂直堆叠的双偶极子阵列（如果是正方形 Quad）。
* **方向图**：
    * 最大辐射方向：**垂直框面 (Broadside)**。
    * 形象比喻：“哑铃”穿过框心，像风扇吹风。
* **优势**：
    * 增益比偶极子稍高（约 3.3 dBi）。
    * 输入阻抗约 100Ω，易于匹配。
    * 闭合环路对静电噪声有抑制作用，接收更安静。

### 4. 总结对比
| 特性 | 电小环 ($C \ll \lambda$) | 全波环 ($C \approx \lambda$) |
| :--- | :--- | :--- |
| **最大辐射方向** | **沿着框面** | **垂直框面** |
| **方向图形状** | “甜甜圈”套在框上 | “哑铃”穿过框心 |
| **物理等效** | 磁偶极子 | 双偶极子阵列 |
